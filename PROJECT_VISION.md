# Trading-RnD-AI

## Vision

Trading-RnD-AI is an autonomous research platform for algorithmic trading.

The system will be capable of:

- Searching for trading strategies from multiple sources
- Understanding strategy logic
- Generating Expert Advisors
- Compiling MQL5 projects
- Fixing compilation errors automatically
- Running backtests
- Evaluating performance
- Improving strategies using AI
- Managing strategy evolution
- Publishing successful products

The goal is not to create one Expert Advisor.

The goal is to build an AI research engineer.
